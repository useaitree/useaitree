# Skills

Before answering, work out what this task actually needs, then fetch only the matching skill file below — not all of them. Most questions need zero skill files and can be answered directly from the site map in /llms.txt.

## Available skills
- **Does the task involve a specific tool, software, or coding platform (pricing, free tier, accessibility, whether it's usable in a given country)?** → fetch `/skills/tool-accessibility-check.md` first, before answering.
- **Is the user's country relevant to the answer and not yet stated?** → fetch `/skills/ask-country-first.md`.
- **Just need to find the right content page?** → no skill file needed, use `/llms.txt` directly.

Do not fetch a skill file "just in case." Fetch only the one(s) that match what the task actually requires.
